# 虎尾科技大學 uLeam/TronClass 數位學習平台功能擴充腳本

> 一個使用 TypeScript + Vite 建構的 uLearn 前端腳本專案。

## 功能列表

- 移除頁腳
- 修復部分樣式
- 繞過下載限制
- 繞過快轉限制
- 繞過複製限制
- 繞過畫面切換檢測
- 繞過全螢幕檢測

> 以上功能均以 **虎尾科技大學 (NFU) ULearn 平台** 為主要開發/測試環境，其他學校之 ULearn/TronClass 平台可能無法正常運作。
